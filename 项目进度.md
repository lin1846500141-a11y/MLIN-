## 💻 回家怎么跑起来(本地开发)
需要装 **Python 3.10+**(前端不需要 Node,除非要上 React)。
```powershell
cd miku-wiki
pip install -r requirements.txt
uvicorn server:app --reload
# 打开 http://127.0.0.1:8000
```
本地没配 DATABASE_URL 时,留言自动存本地 `messages.json`,不影响开发。
